TM-Algo - Take Control of Your Algo Platform

TM-Algo is an open-source, Flask-based Python application designed to bridge the gap between traders and major trading platforms such as Amibroker, Tradingview, Excel, and Google Spreadsheets. With a focus on simplifying algotrading, TM-Algo facilitates easy integration, automation, and execution of trading strategies, providing a user-friendly interface to enhance trading performance.

Getting Started with TM-Algo

Before we begin, ensure you have the following:

  • Visual Studio Code (VS Code) installed on Windows.
  • Python version 3.10 or 3.11 installed.
  • Git for cloning the repository (Download from https://git-scm.com/downloads).
  • PostgreSQL for Windows (current version 16).

Follow these steps to set up your TM-Algo environment:

  1. Install VS Code Extensions: Open VS Code, navigate to the Extensions section on the left tab, and install the Python, Pylance, and Jupyter extensions.
  2. Clone the Repository: Open the VS Code Terminal and clone the TM-Algo repository with the command: git clone https://github.com/marketcalls/TM-Algo
  3. Install Dependencies: Navigate to the directory where TM-Algo is cloned and execute: pip install -r requirements.txt to install the necessary Python libraries.
  4. Set Up PostgreSQL: Download and install PostgreSQL for Windows. Remember the password you set during installation; the default username is `postgres`. Use pgAdmin 4 to create a localhost server and a new database named `TM-Algo_db`.
  5. Configure Environment Variables: Update the `.env` file located in `/TM-Algo/api/` with your specific configurations as shown in the provided template.
  6. Run the Application: From the `/TM-Algo/` directory, start the Flask application with the command: python app.py

After completing the setup, access the TM-Algo platform by navigating to http://127.0.0.1:5000 in your web browser. Log in using the credentials you defined in the `.env` file.

Congratulations! You have successfully set up TM-Algo. Explore the platform and start maximizing your trading performance through automation.
